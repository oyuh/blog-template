## Usage

### GitHub Repo Links

Use the template-safe wrapper:

```astro
<GitHubRepoCardOptional repo="owner/repo" />
<GitHubRepoCardOptional repo="owner/repo" compact />
```

### Link Preview

```tsx
// Hover preview (tooltip on hover)
<LinkPreview url="https://example.com" client:load>Link Text</LinkPreview>

// Always show card
<LinkPreview url="https://example.com" showPreview client:load />
```

### Charts

```tsx
<Chart
  client:load
  data={[
    { x: 'Label 1', y: 10 },
    { x: 'Label 2', y: 20 }
  ]}
  type="line" // or "bar" or "scatter"
  title="Chart Title"
  xLabel="X Axis Label"
  yLabel="Y Axis Label"
  width={600}
  height={400}
/>
```
