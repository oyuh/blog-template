This post demonstrates the interactive components available for blog posts. The charts and
tables are powered by [Chart.js](https://www.chartjs.org), themed to match the site, and stretch
to the full width of the article.

## Charts

Charts are responsive, support multiple datasets, and ship with a small control bar: switch
chart type, toggle a series from the legend, and flip the data into a table.

### Single series

The simplest case — one dataset. The original `data={[...]}` API still works:

<Chart
  client:load
  data={[
    { x: 'Jan', y: 30 },
    { x: 'Feb', y: 45 },
    { x: 'Mar', y: 38 },
    { x: 'Apr', y: 52 },
    { x: 'May', y: 65 },
    { x: 'Jun', y: 58 }
  ]}
  type="line"
  title="Monthly Visitors"
  xLabel="Month"
  yLabel="Visitors (thousands)"
/>

### Multiple datasets + overlay compare

Pass a `series` array to plot several datasets together. With two series visible, the
**Δ Compare** toggle shades the gap between them so the difference is easy to read. Switch
the type to *Bar* or *Area* with the control on the left.

<Chart
  client:load
  type="line"
  title="Weekly visitors by source"
  xLabel="Week"
  yLabel="Visitors"
  height={360}
  series={[
    { name: 'Search', data: [
      { x: 'W1', y: 420 }, { x: 'W2', y: 510 }, { x: 'W3', y: 480 },
      { x: 'W4', y: 620 }, { x: 'W5', y: 700 }, { x: 'W6', y: 660 }
    ]},
    { name: 'Social', data: [
      { x: 'W1', y: 180 }, { x: 'W2', y: 240 }, { x: 'W3', y: 300 },
      { x: 'W4', y: 260 }, { x: 'W5', y: 340 }, { x: 'W6', y: 420 }
    ]}
  ]}
/>

## Data tables

Every chart can flip to a sortable table with the **View as table** button. You can also drop a
standalone `DataTable` anywhere — click a header to sort:

## Usage

### Charts

```tsx
// Single dataset (legacy API)
<Chart
  client:load
  data={[{ x: 'Label 1', y: 10 }, { x: 'Label 2', y: 20 }]}
  type="line" // "line" | "bar" | "area" | "scatter"
  title="Chart Title"
  xLabel="X Axis"
  yLabel="Y Axis"
  height={340}
/>

// Multiple datasets
<Chart
  client:load
  type="bar"
  series={[
    { name: 'Series A', data: [{ x: 'Q1', y: 3 }, { x: 'Q2', y: 5 }] },
    { name: 'Series B', data: [{ x: 'Q1', y: 6 }, { x: 'Q2', y: 4 }] }
  ]}
  barMode="grouped" // or "stacked"
  smooth             // smooth line/area curves
/>
```

Useful props: `controls={false}` hides the control bar, `enableTable={false}` removes the table
toggle, and `allowedTypes={['line','bar']}` limits the type switcher.

### Data tables

```tsx
<DataTable
  client:load
  caption="Optional caption"
  columns={[
    { key: 'name', label: 'Name' },
    { key: 'count', label: 'Count', numeric: true }
  ]}
  rows={[{ name: 'A', count: 12 }, { name: 'B', count: 7 }]}
/>
```
